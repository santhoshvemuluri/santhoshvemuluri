### 👨‍💻 About Me

I'm an engineering student based in Kakinada, Andhra Pradesh, passionate about building robust applications and tackling real-world problems. When I'm not coding, you can find me analyzing Nifty 50 charts for 1-minute scalping setups, hitting the gym with a high-protein focus, or restoring my Yamaha FZ-S V2 and RX 100.

- 🎓 Currently pursuing my B.Tech at **Pragati Engineering College**.
- 💻 Building projects like a **Secure Identity Portal** (JWT/Bcrypt), a **Real-Time Earthquake Detector**, and a **Full-Stack E-commerce Web App**.
- 🌱 Always learning more about **Java**, **React**, and **Embedded Systems**.
